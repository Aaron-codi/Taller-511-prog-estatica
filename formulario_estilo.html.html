Respuestas de Teoria

2-A CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje que se usa para dar estilo y diseño a los documentos web, como los creados con HTML. Diferencias Estilos en linea: Se aplica un estilo directamente a un elemento HTML específico mediante el atributo style, util para aplicacion rapida y directa. Estilos en cabecera: Se escribe dentro de la etiqueta "style" en la sección "head" de un documento HTML, util para un estilo único para una sola página. Estilos mediante externo: Util para proyectos grandes, ademas de poder reutilizar el codigo en multiples paginas y organizar bien el codigo. 2-B El selector de etiqueta selecciona todos los elementos de un tipo específico de etiqueta HTML. El selector de clase se utiliza para seleccionar todos los elementos que tienen una clase específica. Ademas, permiten aplicar el mismo estilo a varios elementos El selector de ID se utiliza para seleccionar un único elemento que tenga un ID específico. Los IDs son únicos en un documento HTML, lo que significa que un ID debe aplicarse solo a un elemento.
