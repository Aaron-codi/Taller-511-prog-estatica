CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje utilizado para definir la apariencia y el diseño de documentos web escritos en HTML o XML. Permite controlar aspectos visuales como colores, fuentes, tamaños, márgenes, espaciado, diseño de cajas, animaciones y más. CSS es esencial para separar el contenido (HTML) de la presentación (estilo), lo que facilita la creación y mantenimiento de sitios web.

Los estilos en linea se aplican directamente sobre los elementos HTML usando el atributo style (esto hablando dentro del HTML), lo que hace rapidos pero poco reutilizables y dificiles de antener. En cambio los estilos externos se guardan en archivos .css separados y se enlazan al HTML, lo que facilita su reutulizacion, mantenimiento separacion de contenido
